# Presenca MVP

Static GitHub Pages deployment for the New Way attendance MVP.

Published files:

- `index.html` - self-contained MVP page
- `.nojekyll` - disables Jekyll processing on GitHub Pages

Deployment notes:

- This repo is intended to stay public and static-only.
- The page is self-contained except for the Google Fonts stylesheet used for `Space Grotesk`.
- No API keys, server code, or private data files are included.
